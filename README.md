# With Socio - Creative Agency Website

Premium video production, personal branding, and web development services.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```

## Build for Production

```bash
npm run build
npm start
```
